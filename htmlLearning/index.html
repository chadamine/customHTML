Marking Elements with Tags

Tags (< />) mark the presence of an element.
Two sided tabs are used for elements with content, and have an opening and closing tag.

Two-sided Tag Syntax:
<element>content</element>

One-sided Tag Syntax:
<element />

<top>


The Structure of an HTML Document

Markup languages require a root element containing all other elements.
For HTML, the root element is marked using the <html> tag:

<html>
 document content
</html>

There are two main sections to an HTML doc:
Head Element
information about the document
Body Element
content that appears in the Web page
